# GoingTour 前端设计方案

## 🎨 整体架构设计

### 技术栈
- **框架**: Vue 3 + JavaScript (用户偏好)
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **UI组件**: Element Plus
- **地图**: Leaflet.js
- **HTTP请求**: Axios
- **构建工具**: Vite

## 📱 页面结构设计

### 1. 用户模块页面 (已有后端支持)

#### 1.1 登录页面 (`/login`)
**功能需求:**
- 手机号输入框（带格式验证）
- 验证码输入框
- "发送验证码"按钮（倒计时功能）
- "登录"按钮
- "还没有账号？去注册"链接

**API集成:**
```javascript
// 发送验证码
POST /api/user/send-code
{
  "phone": "13800138001",
  "type": "login"
}

// 用户登录
POST /api/user/login
{
  "phone": "13800138001", 
  "code": "123456"
}
```

#### 1.2 注册页面 (`/register`)
**功能需求:**
- 手机号输入框
- 验证码输入框
- 昵称输入框（可选）
- "发送验证码"按钮
- "注册"按钮
- "已有账号？去登录"链接

#### 1.3 个人中心页面 (`/personal`)
**功能需求:**
- 用户头像显示/上传
- 昵称编辑
- 手机号显示
- 偏好设置入口
- 我的行程列表
- 退出登录

**API集成:**
```javascript
// 获取用户信息
GET /api/user/{id}

// 更新基本信息
PUT /api/user/{id}/info
{
  "nickname": "新昵称",
  "avatar": "头像URL"
}
```

#### 1.4 偏好设置页面 (`/preferences`)
**功能需求:**
- 旅行偏好标签选择（多选框）
  - ["历史古迹", "自然风光", "美食", "亲子游", "文艺青年", "主题公园"]
- 日均预算滑块（100-2000元）
- "保存设置"按钮

**API集成:**
```javascript
// 更新偏好设置
PUT /api/user/{id}/preferences
{
  "preferences": "[\"历史古迹\", \"美食\"]",
  "budget": 800
}
```

### 2. 行程模块页面 (需要后端API)

#### 2.1 目的地选择页面 (`/destinations`)
**功能需求:**
- 顶部搜索框（城市/景点搜索）
- 热门城市卡片网格
  - 北京卡片（名称+图片+简介）
  - 上海卡片
- 点击卡片进入行程生成页

**需要的后端API:**
```javascript
// 获取城市列表
GET /api/cities

// 搜索城市/景点  
GET /api/search?keyword=北京
```

#### 2.2 行程生成页面 (`/trip/create`) ⭐ 核心页面
**布局设计:**
```
┌─────────────────────────────────────────────────────────┐
│ 顶部控制栏                                               │
│ [目的地: 北京] [天数: 3天] [生成行程按钮]                   │
├────────────────┬────────────────────────────────────────┤
│ 左侧地图区域     │ 右侧时间轴区域                           │
│ Leaflet地图      │ 第1天                                  │
│ - 景点标记       │ ├─ 上午: 天安门广场 [拖拽手柄]           │
│ - 点击显示详情   │ ├─ 下午: 故宫博物院                     │
│                │ └─ 晚上: [添加餐厅]                     │
│                │ 第2天                                  │
│                │ ├─ 上午: [空]                          │
│                │ ├─ 下午: [空]                          │
│                │ └─ 晚上: [空]                          │
└────────────────┴────────────────────────────────────────┘
```

**核心功能:**
- Leaflet地图集成，显示选定城市
- 景点标记点击显示详情弹窗
- 时间轴组件支持拖拽排序
- "生成行程"按钮基于用户偏好自动规划
- 距离计算和交通提示

**需要的后端API:**
```javascript
// 获取城市景点列表
GET /api/attractions?city=北京

// 获取城市餐厅列表  
GET /api/restaurants?city=北京

// 生成个性化行程
POST /api/trips/generate
{
  "userId": 1,
  "city": "北京", 
  "days": 3
}
```

#### 2.3 行程详情页面 (`/trip/{id}`)

## 🔄 路由设计 (`router/router.js`)

```javascript
const routes = [
  // 用户模块
  { path: '/', redirect: '/login' },
  { path: '/login', component: Login },
  { path: '/register', component: Register },
  { path: '/personal', component: Personal, meta: { requiresAuth: true } },
  { path: '/preferences', component: Preferences, meta: { requiresAuth: true } },
  
  // 行程模块  
  { path: '/destinations', component: Destinations, meta: { requiresAuth: true } },
  { path: '/trip/create', component: TripCreate, meta: { requiresAuth: true } },
  { path: '/trip/:id', component: TripDetail, meta: { requiresAuth: true } },
  { path: '/trip/:id/share', component: TripShare },
  
  // 数据模块
  { path: '/attraction/:id', component: AttractionDetail },
  { path: '/search', component: Search }
]
```
